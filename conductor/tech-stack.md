# Tech Stack

- **Frontend:** HTML5, CSS3, Vanilla JavaScript (ES6+).
- **APIs Nativas:** Broadcast Channel (Sincronização), Canvas API (Desenho), LocalStorage (Persistência).
- **External Libraries (Internalized):** QRious v4.0.2, Canvas-Confetti v1.6.0.
- **Architecture:** Standalone Single-Page Application (SPA) com Offline-First approach.
