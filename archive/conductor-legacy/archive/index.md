# 🎨 Design Refinement & Visual Critique (v3.0)

## 🎯 Objetivo
Realizar uma revisão crítica e exaustiva do design visual, interface (UI) e experiência gráfica (UX) do Quadro Branco, explorando novas propostas estéticas para cada funcionalidade.

## 🔍 Análise Crítica Atual
- **Glassmorphism:** Atualmente consistente, mas pode beneficiar de maior contraste em certos temas.
- **Tipografia:** Montserrat é robusta, mas o escalonamento dinâmico (`clamp`) pode ser afinado para textos muito longos.
- **Interação:** Feedback visual de cliques e hover pode ser mais orgânico.
- **Identidade:** Harmonizar as cores das marcas (CML, Futuro Digital, IA) com os temas personalizados.

## 🛠️ Exploração de Melhorias (Index-Temp)

### 1. Sistema de Votação (Polls)
- [ ] Explorar barras com gradientes dinâmicos que mudam de cor conforme a percentagem.
- [ ] Adicionar micro-interações (partículas ou brilho) no vencedor.

### 2. Ink Overlay (Desenho)
- [ ] Implementar "Smoothing" nos traços (Bézier curves) para um aspeto mais profissional.
- [ ] Propor uma paleta de cores "Neon Glow" para o modo escuro.

### 3. Painel de Controlo & Toolbar
- [ ] Testar layouts de "Floating Island" em vez de barra fixa.
- [ ] Refinar ícones para um estilo mais minimalista e unificado.

### 4. Temas e Marcas
- [ ] Criar transições suaves (fade/morph) entre temas.
- [ ] Melhorar o contraste dos logótipos institucionais em temas claros/escuros.
